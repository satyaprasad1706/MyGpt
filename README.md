# MyGPT

A simple chat interface powered by Google's Gemini API and built with Streamlit.

## Features

- Interactive chat interface
- Conversation history
- Powered by Gemini 3 Flash Preview model

## Installation

```bash
pip install streamlit requests
```

## Usage

```bash
streamlit run MyGpt.py
```

## Configuration

Replace the API key in the code with your own Google Gemini API key.

## Requirements

- Python 3.7+
- streamlit
- requests
